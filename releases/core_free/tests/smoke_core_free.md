# Core Free Smoke Checklist

1. Start app with a clean app home and the `core_free` release overlay.
2. Confirm plugin package install/update/remove UI is not available.
3. Confirm `Management` and `Service` stages stay compatible with the fixed pipeline but show product empty-state messaging instead of bundled plugins.
4. Confirm transcription provider defaults to `transcription.whisperadvanced`.
5. Confirm LLM provider defaults to `llm.llama_cli`.
6. Confirm semantic processing plugins are discoverable and enabled.
7. Confirm app starts even when no Whisper ASR model is present.
8. Confirm `Settings > Transcription` shows the Whisper model list and the user can choose a model and download it.
9. Confirm app starts even when no llama model is present.
10. Confirm Settings show first-run local-model onboarding for the LLM provider.
11. Confirm Settings offer all three LLM model setup paths:
   - download a llama model from a list
   - provide a custom direct `.gguf` URL
   - select a local GGUF file
12. Download a Whisper model, run transcription, and confirm transcript output is produced.
13. Run pipeline without a llama model and confirm semantic processing still produces useful output.
14. Confirm the edited/minutes output is generated by semantic processing even when `llm.llama_cli` falls back because no model is configured.
15. Configure or download a llama model, rerun processing, and confirm local LLM output is produced without changing the release profile.
